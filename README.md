# -日报skills-让Agent秀-
帮你一键整理，今天你的Agent都做什么了
其实这个skills的英文名字叫let agent cook，我很喜欢。
然后我的设计想法是，每天读取claude code的上下文内容，然后帮你总结协作日报。
当天的核心点是什么，核心的三个产出是什么，最好的实践是什么，最差的实践是什么，最好的提示词是什么。
如果有展示内容，两个展示demo是什么。
然后一键生成一个本地网页。左右两边自动生成一些代码，是当天Agent执行的任务。
